Core C-NHEJ machinery:

- XRCC6 (Ku70) — DSB end recognition
- XRCC5 (Ku80/Ku86) — DSB end recognition, heterodimer with Ku70
- PRKDC (DNA-PKcs) — synapsis, kinase signaling
- NHEJ1 (XLF/Cernunnos) — ligation scaffold/bridging
- XRCC4 — stabilizes and stimulates LIG4
- LIG4 (DNA Ligase IV) — the ligase
- PAXX (C9orf142) — paralog of XRCC4/XLF, end stabilization
- DCLRE1C (Artemis) — endonuclease, end processing

End processing / accessory factors:

- POLL (Pol λ) and POLM (Pol μ) — fill-in polymerases
- DNTT (TdT) — template-independent nucleotide addition (lymphoid)
- PNKP — polynucleotide kinase/phosphatase, end cleanup
- APLF — aprataxin-PNK-like factor, scaffolding
- APTX (aprataxin) — deadenylation of abortive ligation
- TDP1, TDP2 — tyrosyl-DNA phosphodiesterases
- WRN — helicase/exonuclease processing
- CYREN (MRI/C7orf49) — cell-cycle-dependent NHEJ modulator

The MRN complex (MRE11, RAD50, NBN) shows up in some NHEJ gene lists for end tethering/processing, but it's primarily an HR/resection complex — worth flagging as ambiguous depending on how strictly you define the pathway boundary.
